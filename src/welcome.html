Welcome to librarymanagementsystem-analytics Service

Produces personal, branch, and admin analytic dashboards; leverages MongoDB aggregation & change streams for real-time and historical analytics; generates usage, trend, and audit reports; stores audit trails; presents localized, personalized, and global analytics; supports compliance.

Available Routes

  • getAnalyticSnapshot => GET /analyticsnapshots/:analyticSnapshotId
  • createAnalyticSnapshot => POST /analyticsnapshots
  • updateAnalyticSnapshot => PATCH /analyticsnapshots/:analyticSnapshotId
  • deleteAnalyticSnapshot => DELETE /analyticsnapshots/:analyticSnapshotId
  • listAnalyticSnapshots => GET /analyticsnapshots
  • getAuditLog => GET /auditlogs/:auditLogId
  • createAuditLog => POST /auditlogs
  • updateAuditLog => PATCH /auditlogs/:auditLogId
  • deleteAuditLog => DELETE /auditlogs/:auditLogId
  • listAuditLogs => GET /auditlogs
  • getChangeStreamEvent => GET /changestreamevents/:changeStreamEventId
  • createChangeStreamEvent => POST /changestreamevents
  • deleteChangeStreamEvent => DELETE /changestreamevents/:changeStreamEventId
  • listChangeStreamEvents => GET /changestreamevents
